# User Safety

**Detta dokument är en sammanfattning. För fullständig information, se [security-complete.md](security-complete.md#user-safety).**

## Översikt

Copy/Paste-systemet skyddar användare från oavsiktlig dataförlust och säkerställer att destruktiva operationer kräver explicit bekräftelse.

## Huvudprinciper

1. **Original Lock** - Original transcript får aldrig redigeras direkt
2. **Dry-Run som Default** - Farliga operationer körs i dry-run-läge som standard
3. **Receipt System** - Alla destruktiva handlingar returnerar receipt för spårbarhet
4. **Human-in-the-Loop** - Inga autonoma beslut, alltid mänsklig kontroll
