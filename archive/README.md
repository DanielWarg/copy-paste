# Archive

Detta är en arkivmapp för kod och dokumentation som inte längre är aktivt använd i projektet men som kan vara värdefullt att behålla för referens.

## Innehåll

### modules/

Inaktiva moduler som inte längre är registrerade i `main.py`:
- `drafting/` - Drafting module (under utveckling/planerad)
- `ingestion/` - Ingestion module (under utveckling/planerad)
- `privacy/` - Privacy module (legacy, ersatt av privacy_shield)
- `privacy_v2/` - Privacy V2 module (under utveckling/planerad)

**Notera:** Dessa moduler kan återaktiveras i framtiden om behovet uppstår.

### docs/

Utdaterade planeringsdokument:
- `ARBETSPLAN.md` - Gammal projektplan
- `SCOUT_PLAN.md` - Scout module plan
- `projektplan.md` - Projektplan

**Notera:** Aktuell projektstatus finns i `todo.md` och `CHANGELOG.md`.
